# Chrome Web Store 描述文档

## 扩展名称
Smart Bookmark Extension - 智能书签管理器

## 简短描述
基于活跃度排序和快速搜索的智能书签管理扩展，让收藏网页变得更简单高效。

## 详细描述
Smart Bookmark Extension 是一款革命性的书签管理工具，旨在提升您的书签组织和访问效率。

### 核心功能

**智能排序**
- 根据文件夹活跃度自动排序书签文件夹
- 活跃度基于文件夹中书签的数量和使用频率
- 最常用的文件夹始终显示在顶部

**快速搜索**
- 实时模糊搜索功能
- 智能匹配算法（完全匹配 > 前缀匹配 > 包含匹配）
- 搜索响应时间小于100ms

**快捷操作**
- 使用 Ctrl+Shift+B 快捷键快速收藏当前页面
- 简洁直观的用户界面
- Modal打开时间小于200ms

**性能优化**
- 智能缓存机制，减少重复API调用
- 内存占用小于50MB
- 防抖搜索，减少不必要的计算

### 使用方法
1. 在任何网页上按下 Ctrl+Shift+B（Windows/Linux）或 Cmd+Shift+B（Mac）
2. 在弹出的对话框中搜索或浏览文件夹
3. 选择目标文件夹
4. 点击"添加书签"完成操作

Smart Bookmark Extension 尊重您的隐私。我们不会收集、存储或分享您的任何个人信息。扩展仅使用必要的浏览器权限来实现书签管理功能。
