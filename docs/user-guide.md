# Smart Bookmark Extension 用户使用指南

## 简介

Smart Bookmark Extension 是一款智能书签管理扩展，可帮助您更高效地组织和访问书签。通过智能排序和快速搜索功能，您可以轻松将网页收藏到最相关的文件夹中。

## 安装

1. 下载扩展包
2. 打开 Chrome 浏览器
3. 访问 `chrome://extensions/`
4. 启用"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择扩展文件夹

## 使用方法

### 快速收藏网页

1. 在任何网页上点击扩展图标，或使用你在快捷键页面配置的快捷键（默认未设置）
2. 在弹出的对话框中，您可以：
   - 搜索文件夹（输入关键词过滤文件夹列表）
   - 浏览文件夹（按活跃度排序的文件夹列表）
3. 点击选择目标文件夹
4. 点击"添加书签"按钮

### 文件夹排序

文件夹按活跃度自动排序，活跃度基于文件夹中书签的数量。最常用的文件夹会显示在列表顶部。

### 搜索功能

在搜索框中输入关键词，系统会实时过滤文件夹列表：
- 完全匹配（最高优先级）
- 前缀匹配
- 包含匹配

## 快捷键

- 默认未设置快捷键，请手动配置

### 自定义快捷键

您可以自定义扩展的快捷键：

1. 在浏览器地址栏输入 `chrome://extensions/shortcuts` 并回车
2. 找到 "Smart Bookmark Extension" 下的“打开智能书签”命令
3. 点击当前快捷键设置，在弹出框中输入您想要的快捷键组合
4. 点击"确定"保存设置

注意：某些快捷键组合可能与浏览器或其他扩展冲突，建议使用 `Ctrl/Cmd+Shift+字母` 的组合。

## 性能特点

- 响应迅速：Modal打开时间小于200ms
- 搜索快速：搜索响应时间小于100ms
- 内存优化：内存占用小于50MB
- 智能缓存：减少重复操作的API调用

## 故障排除

### 扩展不工作

1. 确认扩展已启用
2. 刷新页面
3. 重启浏览器

### 书签未保存

1. 检查是否选择了正确的文件夹
2. 确认网络连接正常
3. 查看浏览器控制台是否有错误信息

### 搜索结果不准确

1. 尝试使用更具体的关键词
2. 检查书签文件夹名称是否包含关键词

## 常见问题

### 为什么文件夹排序会变化？

文件夹按活跃度排序，活跃度基于文件夹中书签的数量。当您向文件夹添加新书签时，该文件夹的活跃度会增加，排序位置可能会上升。

### 扩展会访问我的个人数据吗？

不会。本扩展只使用书签和当前页面URL信息，不收集或传输任何个人数据。

### 如何卸载扩展？

1. 访问 `chrome://extensions/`
2. 找到 "Smart Bookmark Extension"
3. 点击"移除"按钮
